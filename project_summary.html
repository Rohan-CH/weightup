Purpose of the App

WeightUp is a comprehensive fitness application designed to help gym-goers and fitness enthusiasts systematically track their strength training. Instead of relying on messy physical notebooks, users can digitally log their exercises, visualize their strength progression over time, and stay motivated by engaging with a global community leaderboard.

Key Features

  • Intuitive Workout Logging
    Quickly record the exercise type, weight lifted, and number of repetitions on specific dates.
  • Interactive Progress Charts
    Visualize strength gains across individual exercises using dynamic, easily readable line charts.
  • Community Leaderboard
    Foster motivation and healthy competition by comparing personal bests and activity with other users globally.
  • Secure Profiles & Customization
    Create custom exercises, upload personal avatars, and maintain a private, secure training history.

Technology Stack

WeightUp is built using modern, industry-standard web technologies to ensure a fast, secure, and scalable experience.

Next.js & React
The Frontend is powered by React 19 and Next.js 16, providing a lightning-fast, highly responsive user interface that feels like a native app.
Supabase
The Backend & Database rely on Supabase (PostgreSQL). It securely handles user accounts, stores all workout data, and hosts profile images.
Tailwind CSS
Used for Styling. It allowed for the rapid creation of a beautiful, custom dark-mode aesthetic that looks great on any screen size.
Recharts
A specialized Data Visualization library used to draw the dynamic progress graphs based on the user's logged workout data.
TypeScript
The entire app is written in TypeScript, adding a layer of security to the code to prevent bugs before they happen.
